# TypeSwift 
TypeSwift is a simple and fun app to test your typing speed and accuracy.

## Features
- Typing speed test (measures WPM and accuracy)
- Real-time performance feedback

## Try It Out
Check it out at typeswift.vercel.app and start typing!
